# TenBuddy

一个 PC 桌面聊天助手，目前仅用于研究学习，看看 LLM 的能力在 IM 场景里能做什么。

## 功能

- 支持 Windows 版本的 PC WeChat ✅
- 支持 Windows 版本的 QQ [未实现]
- 支持 MacOS 版本的 PC WeChat [未实现]
